BlueStark Simple Static Website

Purpose:
A simple independent web presence for www.bluestarkonline.com.
Afriophthalmics is intentionally not referenced as the parent brand.

Files:
- index.html
- styles.css
- script.js
- assets/images/*
- assets/icons/*

How to preview locally:
Open index.html directly in a browser.

How to deploy simply:
1. Buy/connect the domain www.bluestarkonline.com with your hosting provider.
2. Upload all files in this folder to the public_html / root web directory.
3. Make sure index.html is at the root.

Vercel deployment:
1. Create a GitHub repository, for example bluestark-online.
2. Upload all files from this folder.
3. Import the repository into Vercel as a static project.
4. Add the domain www.bluestarkonline.com in Vercel > Project Settings > Domains.

Important edits before launch:
- In script.js, replace info@bluestarkonline.com if you want a different enquiry email.
- Replace or remove the request form if you do not want any contact capture.
- Add your real favicon/icon if you create a final parent BlueStark logo without edition labels.

Copy rules followed:
- No “TV-based” wording.
- The screen is referred to as the chart monitor.
- Lite and Premium are treated as product editions.
- BlueStark is treated as an independent brand.
- Visual Field Screener is described as software in development / coming soon.
